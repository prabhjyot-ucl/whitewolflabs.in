## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | Main website (single page) |
| `style.css` | All styles, animations, responsive layout |
| `script.js` | Interactions, cursor, counters, scroll effects |

## ✨ Features

- Custom animated cursor (desktop)
- Hero with animated grid background + floating orbs
- Running ticker marquee
- Animated stat counters
- Scroll reveal animations
- Card tilt effects on hover
- Mobile-responsive + mobile menu
- Contact form with success state
- Rotating "badge" element
- Smooth scroll navigation

## 🌐 Deploy on GitHub Pages

1. Push all files to the `main` branch of `prabhjyot-ucl/whitewolflabs.in`
2. Go to **Settings → Pages**
3. Source: **Deploy from branch → main → / (root)**
4. Your site will be live at `https://prabhjyot-ucl.github.io/whitewolflabs.in/`

To use the custom domain `whitewolflabs.in`:
- Add a `CNAME` file with `whitewolflabs.in`
- Point your domain's DNS to GitHub Pages
